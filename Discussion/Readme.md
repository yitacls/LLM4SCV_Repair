## Discussion Overview

This folder contains supplementary experiments and results reported in the Discussion section of the paper.

## Contents

1. **GPT-4o & GPT-5-mini**
   - Contains experimental results for the **latest GPT model series** added as supplementary evaluation.

2. **GPT-o1 For Contract Vulnerability Repair**
   - Contains experimental results for the **reasoning model o1**.

3. **Pass@k**
   - Contains results showing how the repair performance of **GPT-3.5** changes with **different numbers of calls/samples** (Pass@k-style evaluation).

4. **Prompt Optimization**
   - Contains results showing how the repair performance of **GPT-3.5** varies under **different prompting strategies**.
