## Dataset Overview

This folder contains the datasets used in the experiments, including the original vulnerable contracts, an additional set of high-version vulnerable contracts, and manually repaired contracts.

## Contents

1. **origin**
   - Contains the **original smart contract vulnerability dataset**.

2. **high-level contract.zip**
   - Contains **100 additional high-version vulnerable smart contracts** collected as a supplementary dataset.

3. **Repaired Contract**
   - Contains **103 manually repaired vulnerable smart contracts** (human-written fixes).
